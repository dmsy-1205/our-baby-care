# HearMe2nite

두 사람이 하루를 함께 기록하고, 약속과 미션, 기념일, 운동, 추억을 한 공간에 남기는 커플 라이프 앱입니다.

## 주요 기능
- HearMe2nite 자체 회원가입 및 로그인
- 우리의 공간 생성 / 초대코드 입장 / 기존 공간 복구
- 상대방 온라인 상태 표시
- 오늘의 약속과 커스텀 미션
- 오늘의 컨디션과 하루 기록
- 오늘의 운동 기록
- 우리의 기념일
- 기록실 캘린더와 하루 기록 팝업
- 채팅
- 앱 내부 사용 가이드

## RC2.18 핵심
RC2.18은 문서를 길게 늘리는 버전이 아니라, 사용자가 앱 안에서 바로 이해할 수 있는 짧은 도움말 경험을 만드는 버전입니다.
