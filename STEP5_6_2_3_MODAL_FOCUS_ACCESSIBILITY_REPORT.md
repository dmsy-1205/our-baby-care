# HearMe2nite v1.0 STEP5.6.2.3 변경 보고서

## 수정 목적
모달 닫기 버튼에 포커스가 남은 상태에서 `aria-hidden=true`가 적용되어 발생하던 Chrome 접근성 경고를 제거했습니다.

## 적용 내용
- 공통 모달 열기 시 호출 버튼을 기억합니다.
- 모달 닫기 전에 포커스를 호출 버튼으로 복귀시킵니다.
- 호출 버튼이 없으면 내부 포커스를 안전하게 해제합니다.
- 닫힌 모달에는 `inert`와 `aria-hidden=true`를 함께 적용합니다.
- 열린 모달에서는 `inert`를 먼저 제거한 뒤 포커스를 이동합니다.
- 오늘의 요약, 기록실, 하루 입력, 미션, 프로필 및 사진 갤러리 모달에 공통 적용됩니다.

## 변경하지 않은 항목
- Firebase 데이터 구조
- 저장 및 자동 저장 로직
- Room, 인증, 데이터 관리 기능
- 기존 UI 배치와 사용자 기록
