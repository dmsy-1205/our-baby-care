# HearMe2nite v1.0 STEP5.6.3.2 작업 보고서

## 적용 내용

### 1. 우리의 대화 위치 변경
- `우리의 대화` 카드를 `오늘의 요약` 바로 아래로 이동했습니다.
- 화면 순서는 `기록 날짜 → 오늘의 요약 → 우리의 대화 → 오늘의 약속`으로 고정됩니다.
- 기존 실시간 채팅, 읽지 않은 메시지, 온라인 상태, 채팅 모달 기능은 유지했습니다.

### 2. 오늘의 약속 주간 루틴 관리
- 오늘의 약속 관리 화면에 `오늘의 미션 / 주간 루틴` 유형을 추가했습니다.
- 주간 루틴은 일~토 중 여러 요일을 선택할 수 있습니다.
- `매일` 버튼으로 7개 요일을 한 번에 선택할 수 있습니다.
- 기존 카드는 별도 설정이 없으면 `오늘의 미션`으로 처리하여 호환성을 유지합니다.

### 3. 홈 카드 표시
- 오늘의 약속 메인 카드 아래에 선택한 기록 날짜에 해당하는 약속을 작은 카드로 표시합니다.
- 오늘의 미션은 매일 표시됩니다.
- 주간 루틴은 지정 요일에만 표시됩니다.
- 각 카드에는 미션/요일 정보와 완료 진행 상태를 표시합니다.
- 카드를 누르면 기존 입력 모달이 열립니다.

## 데이터 구조
기존 `rooms/{roomCode}/customCards/{cardId}`를 유지하며 다음 필드만 추가합니다.

- `kind`: `mission` 또는 `weekly`
- `weekdays`: 주간 루틴 요일 배열(0=일요일, 6=토요일)

기존 데이터, 날짜별 입력값, Firebase Rules 및 데이터 관리 기능은 변경하지 않았습니다.
