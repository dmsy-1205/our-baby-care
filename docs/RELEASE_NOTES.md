## 이번 버전 목표

RC2.18은 기능 추가보다 제품 완성도를 위한 문서화와 앱 내부 도움말 구축에 집중한 버전입니다.

## New

- `/docs` 폴더 추가
- 사용자 설명서 추가
- FAQ 추가
- CHANGELOG 추가
- Release Notes 정리
- Architecture 문서 추가
- Project Rules 문서 추가
- UI Guideline 추가
- Deploy Checklist 추가
- Roadmap 추가
- 앱 내부 Help Center 추가

## Improved

- 기존 `사용설명서` 모달을 `도움말 센터`로 확장
- 주요 기능별 안내 섹션 분리
- GitHub README를 사용자 중심으로 개편
- v1.0 Stable 준비를 위한 문서 체계 정리

## Fixed

- 문서와 실제 기능 간 설명 누락 보완
- 운동 기능 추가 이후 사용자 설명서 누락 보완
- 기념일 기능 설명 보완
