# HearMe2nite 사용자 설명서

현재 버전: **v1.0 STEP5.10.9**  
배포일: **2026.07.15**  
단계: **Beta**

> 앱 안의 `?` 버튼을 누르면 검색 가능한 최신 사용자 설명서를 확인할 수 있습니다. 버전·배포일·최근 업데이트 항목은 `js/release-info.js`에서 자동으로 표시됩니다.

## 1. 시작하기

1. 이메일과 비밀번호로 회원가입합니다.
2. 받은 인증 메일의 링크를 눌러 이메일 인증을 완료합니다.
3. 새로운 우리의 공간을 만들거나, 상대방에게 받은 24시간 유효 초대코드로 입장합니다.
4. 관리(Dom) 또는 기록(Sub) 역할을 선택합니다. 역할은 일반 화면에서 임의 변경할 수 없습니다.

## 2. 우리의 공간

현재 공간, 역할, 상대방 접속 상태를 확인합니다. 이전에 참여한 공간이 있다면 이전 공간 목록에서 다시 전환할 수 있습니다. 초대코드는 생성 후 24시간이 지나거나 한 번 사용되면 다시 사용할 수 없습니다.

## 3. 오늘의 약속

관리(Dom)가 약속을 만들고 기록(Sub)이 홈 카드에서 수행합니다. 약속에는 텍스트·체크·숫자·시간 입력 항목을 구성할 수 있습니다. 결과는 기록실과 최종 복사 내용에 저장됩니다.

## 4. 나의 루틴

기록(Sub)이 자신의 하루 루틴을 직접 만들고 관리합니다. 관리(Dom)는 읽기 전용으로 결과를 확인합니다.

- 반복: 매일, 요일 선택, 오늘만, 일시 중지
- 입력 항목: 텍스트, 체크, 숫자, 시간
- 관리: 대표 카드를 눌러 생성·수정·삭제
- 실행: 홈에 생성된 개별 루틴 카드에서 입력 및 완료
- 보존: 날짜별 스냅샷으로 기록실과 최종 복사 결과에 반영

## 5. 오늘의 컨디션과 기록

기분, 운동, 체중, 수분을 기록하고, 기상 시간·식사·외출·취침 예정·오늘의 하루 메모와 사진을 남길 수 있습니다. 관리(Dom)는 관리와 피드백 영역에서 피드백과 선물을 작성합니다.

## 6. 우리의 대화

같은 공간의 두 사용자가 실시간으로 대화합니다. 메시지 시간, 읽음 상태, 상대방 온라인·오프라인 상태를 확인할 수 있습니다.

## 7. 기록실

달력에서 날짜를 누르면 그날의 오늘의 약속, 나의 루틴, 컨디션, 기록, 피드백을 확인합니다. 사진 모아보기와 기념일 관리도 기록실에서 사용할 수 있습니다.

## 8. 테마와 화면 모드

- 개인 테마: 라벤더, 블라썸, 오션, 포레스트, 크림
- 우리의 공용 테마: 관리(Dom)가 켜면 같은 공간의 두 사용자에게 실시간 적용
- 표시 방식: 라이트, 다크, 시스템 설정 따라가기

공용 테마를 끄면 각 사용자의 기존 개인 테마로 돌아갑니다. 표시 방식은 개인별로 저장됩니다.

## 9. 계정과 데이터 관리

홈 상단 계정 카드를 눌러 프로필, 테마, 데이터 관리, 관리자 메뉴, 로그아웃을 사용합니다. 데이터 삭제는 즉시 실행되지 않으며 관리자 검토를 거칩니다. 처리 전에는 요청을 취소할 수 있습니다.

## 10. 이번 업데이트

STEP5.10.9에서는 사용자 설명서를 최신 기능 기준으로 전면 개편하고, 홈 버전·브라우저 제목·설명서 업데이트·QA 버전을 `js/release-info.js` 한 곳에서 자동 동기화하도록 변경했습니다.

## 11. 버전 운영 원칙

새 버전을 만들 때 다음 항목을 반드시 수행합니다.

1. `js/release-info.js`의 버전, 날짜, 제목, 변경 목록 수정
2. 새 기능의 상세 사용법을 앱 도움말과 이 문서에 반영
3. QA 콘솔의 `RELEASE_SYNC` 결과 확인
4. 전체 소스 ZIP 무결성 검사 후 배포
