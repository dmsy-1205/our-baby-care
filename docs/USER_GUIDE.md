# HearMe2nite 사용자 설명서

HearMe2nite는 두 사람이 하루의 컨디션, 약속, 운동, 기념일, 사진, 기록, 채팅을 하나의 안전한 공간에서 함께 관리하는 앱입니다.

## 1. 시작하기

1. HearMe2nite에 로그인합니다.
2. 승인된 계정인지 확인합니다.
3. `우리의 공간`을 엽니다.
4. 혼자 사용할 공간을 만들거나 상대방 초대코드로 같은 공간에 입장합니다.
5. 오늘의 컨디션과 기록을 작성합니다.

## 2. 우리의 공간

`우리의 공간`은 앱의 모든 기록이 저장되는 Room입니다. 공간을 만들기 전에는 오늘의 기록, 기록실, 채팅이 정상적으로 표시되지 않을 수 있습니다.

### 혼자 시작

`우리의 공간` → `새로운 공간 시작하기`를 누르면 혼자 바로 기록할 수 있습니다. 상대방은 나중에 초대할 수 있습니다.

### 함께 시작

1. 한 사람이 먼저 공간을 만듭니다.
2. `상대방 초대하기`에서 초대코드를 생성합니다.
3. 상대방에게 초대코드를 전달합니다.
4. 상대방은 `받은 초대코드 입력`에서 코드를 입력합니다.

## 3. 역할 선택

- `관리(Dom)`: 미션, 피드백, 보상, 휴식, 관리 메모 중심
- `기록(Sub)`: 하루 기록, 컨디션, 운동, 피드백 확인 중심

역할은 최초 1회 선택 후 일반 화면에서는 변경되지 않도록 잠깁니다.

## 4. 오늘의 컨디션

홈 화면의 오늘의 컨디션에서는 다음 항목을 기록합니다.

- 기분
- 기상시간
- 운동
- 체중
- 수분

각 카드를 누르면 입력 팝업이 열립니다. 입력한 내용은 실시간 저장됩니다.

## 5. 오늘의 기록

하루 동안 있었던 일, 식사, 외출, 취침, 사진, 자유 기록을 남길 수 있습니다. 최종 복사 기능을 통해 하루 내용을 정리해 공유할 수 있습니다.

## 6. 오늘의 운동

운동 카드는 RC2.17.3에서 추가된 기능입니다. 오늘 한 운동, 강도, 시간, 느낌을 기록하고 History와 최종 복사 내용에 함께 반영합니다.

## 7. 우리의 기념일

생일, 여행, 데이트, 휴가, 사용자 지정 기념일을 등록할 수 있습니다. 기념일은 캘린더에 아이콘으로 표시됩니다.

향후 개선 예정 항목은 다가오는 순 정렬, 대표 날짜 지정, 함께한 지 n일 표시, 카드 최대 3개 표시, 더보기 및 관리 모달입니다.

## 8. 기록실

기록실은 저장된 하루 기록을 다시 보는 공간입니다.

- Calendar에서 날짜를 클릭합니다.
- 해당 날짜의 하루 기록 팝업을 확인합니다.
- 사진 모아보기로 저장된 이미지를 확인합니다.

## 9. 채팅

같은 Room에 연결된 사용자끼리 실시간 채팅을 사용할 수 있습니다. 방이 없거나 상대가 같은 방에 없으면 채팅이 공유되지 않습니다.

## 10. 데이터 복구

`우리의 공간` → `이전 공간 불러오기`에서 기존 Room을 복구할 수 있습니다. 직접 방코드 입력이 필요한 경우 수동으로 연결할 수 있습니다.

## 11. 문제가 생겼을 때

먼저 다음을 확인하세요.

- 로그인 상태
- 승인 상태
- 같은 Room 연결 여부
- 인터넷 연결
- Firebase 배포 상태
- Netlify 최신 배포 여부

자세한 문답은 `FAQ.md`를 참고하세요.
