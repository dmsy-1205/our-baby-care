# HearMe2nite STEP6.2.14.47 실제 기기 회귀 검사

## 목적

Dom/Sub 두 세션과 iPhone 홈 화면 PWA에서 자동 검사만으로 확인할 수 없는 실제 동작을 검증한다.

## 사전 조건

- 테스트 앱 `https://hearme2nite1205.web.app`만 사용한다.
- Dom과 Sub는 서로 다른 브라우저 또는 기기를 사용한다.
- 메인 앱과 Firebase Database, Storage, Functions, 규칙은 변경하거나 배포하지 않는다.
- 테스트할 날짜와 Room 코드를 기록한다.

## Dom/Sub 두 세션

- [ ] Dom에서 닉네임을 변경하면 Sub 프로필 바에 같은 이름이 표시된다.
- [ ] Sub에서 닉네임을 변경하면 Dom 프로필 바에 같은 이름이 표시된다.
- [ ] Dom 저장 직후 Sub로 빠르게 전환해도 Dom 닉네임이 Sub 본인 이름으로 기록되지 않는다.
- [ ] Room A에서 프로필 저장 중 Room B로 전환해도 Room B presence에 Room A 이름이 기록되지 않는다.
- [ ] 코멘트 알림을 누르면 원본 기록 카드 없이 코멘트 창만 열린다.
- [ ] 코멘트 창을 한 번 닫으면 홈으로 돌아온다.
- [ ] 일반 기록 카드를 열면 코멘트 버튼이 보인다.
- [ ] 네트워크를 끊고 코멘트 저장을 시도하면 입력 내용이 유지된다.
- [ ] 저장된 코멘트 작성자명이 프로필 닉네임과 일치한다.

## 키보드와 스크린리더

- [ ] 코멘트 창을 열면 닫기 버튼으로 포커스가 이동한다.
- [ ] Tab과 Shift+Tab 포커스가 코멘트 창 안에서 순환한다.
- [ ] Escape로 코멘트 창을 닫을 수 있다.
- [ ] 닫은 뒤 원래 코멘트 버튼 또는 알림 바로 포커스가 돌아온다.
- [ ] VoiceOver가 “함께 남긴 말” 대화상자 제목을 읽는다.

## iPhone 홈 화면 PWA

- [ ] 기존 설치 아이콘에서 앱을 완전히 종료한 뒤 다시 열어 정상 화면이 표시된다.
- [ ] Safari에서 새 버전을 한 번 연 뒤 기존 홈 화면 앱도 새 버전으로 갱신된다.
- [ ] Wi-Fi와 셀룰러를 모두 끄고 실행하면 빈 화면이 아니라 오프라인 안내가 표시된다.
- [ ] 네트워크를 다시 켜고 재실행하면 재설치 없이 정상 복구된다.
- [ ] 앱 전환 후 10분 뒤 돌아와도 로그인 화면 또는 홈 화면이 정상 표시된다.
- [ ] 기기 재부팅 후 홈 화면 아이콘으로 정상 실행된다.

## 고정 화면 크기

- [ ] 360px 라이트/다크
- [ ] 673px 라이트/다크
- [ ] 768px 라이트/다크
- [ ] 1440px 라이트/다크
- [ ] 가로 넘침 없음, 홈 카드 2열 유지, 모달 닫기와 입력 버튼 가림 없음
