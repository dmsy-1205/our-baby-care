🏡 우리의 공간

아직 연결된 공간이 없습니다. 혼자 사용할 공간을 만들거나 초대코드를 입력해 주세요.
👥 내 관계 역할 선택
이 설정은 앱 설치 순서나 방 생성 순서가 아니라, 두 사람의 관계 역할을 의미합니다. 최초 1회 선택 후 일반 화면에서는 변경할 수 없습니다.
처음 공간을 만들거나 초대코드로 참여하기 전에 역할을 선택해 주세요.
🔗 공간 연결과 이동 처음 시작하거나 다른 공간으로 이동할 수 있어요.
💌 상대방 초대하기 초대코드 만들기
공간을 만든 뒤 초대코드를 생성하세요. 코드는 24시간 동안 유효하며 문자, 카톡, 이메일로 보낼 수 있습니다.
🎟️ 받은 초대코드 입력 초대받은 공간 참여
상대방에게 받은 초대코드를 입력하면 같은 공간에 참여하거나 새 공간으로 이동할 수 있습니다.
📂 이전 공간 불러오기 참여했던 공간으로 이동
내가 이미 참여했던 공간 목록을 보여줍니다. Dom과 Sub 모두 버튼으로 이전 공간에 다시 연결할 수 있어요.

기능 선택

👋 처음 사용하는 분께

로그인 후 우리의 공간을 연결하고, 홈에서 오늘의 약속·나의 루틴·컨디션·기록을 차례로 사용하세요.

💡 관리(Dom)는 약속과 피드백을 관리하고, 기록(Sub)은 약속 수행과 자신의 루틴·하루 기록을 작성합니다.

🏠 홈 화면 순서

우리의 공간 → 우리의 대화 → 기록 날짜 → 알림 바 → 우리의 흐름 → 오늘의 요약 순서로 확인한 뒤 오늘의 약속과 기록을 작성합니다.

👤 내 계정

홈 상단의 계정 카드를 누르면 프로필, 테마, 데이터 관리, 관리자 메뉴, 로그아웃을 사용할 수 있습니다.

1️⃣ 회원가입과 인증

이메일과 비밀번호로 가입한 뒤 인증 메일의 링크를 눌러 인증을 완료합니다.

2️⃣ 우리의 공간 만들기

먼저 공간을 만든 사용자는 관리(Dom) 또는 기록(Sub) 역할을 선택합니다. 선택한 역할은 일반 화면에서 임의로 변경할 수 없습니다.

3️⃣ 상대방 초대

초대코드는 생성 후 24시간 동안 사용할 수 있습니다. 상대방은 받은 코드를 입력해 같은 공간에 입장합니다.

🏡 우리의 공간

현재 공간 코드, 내 역할, 상대방의 접속 상태를 확인할 수 있습니다.

🔁 이전 공간 전환

이전에 참여했던 공간이 있다면 우리의 공간 메뉴에서 다시 불러오고 전환할 수 있습니다.

🔐 초대코드 보안

초대코드는 24시간 후 만료되며, 이미 사용된 코드는 다시 사용할 수 없습니다.

💜 오늘의 약속

관리(Dom)가 약속을 만들고, 기록(Sub)이 홈의 약속 카드에서 내용을 입력하거나 완료합니다.

⚙️ 약속 관리

관리 화면에서 텍스트·체크·숫자·시간 항목을 구성하고, 약속을 수정하거나 삭제할 수 있습니다.

📚 기록 보존

입력한 오늘의 약속은 최종 복사 결과와 기록실의 날짜별 기록에 함께 표시됩니다.

🌱 나의 루틴

기록(Sub)이 자신의 하루 일과를 직접 만들고 수행하는 공간입니다. 관리(Dom)는 결과를 읽기 전용으로 확인합니다.

🛠️ 루틴 만들기

나의 루틴 대표 카드를 눌러 새 루틴을 만들고, 매일·요일 선택·오늘만·일시 중지를 설정할 수 있습니다.

📝 루틴 입력 항목

각 루틴에는 텍스트·체크·숫자·시간 항목을 추가할 수 있습니다. 실제 입력은 홈에 생성된 루틴 카드에서 진행합니다.

🗓️ 루틴 기록

완료 상태와 입력 내용은 날짜별로 저장되고, 기록실과 최종 복사 결과에 오늘의 약속 다음 순서로 표시됩니다.

😊 오늘의 컨디션

기분, 운동, 체중, 수분을 기록합니다. 값이 없으면 요약에는 ‘-’로 표시됩니다.

📝 오늘의 기록

기상 시간, 식사, 외출, 취침 예정, 오늘의 하루 메모와 사진을 남길 수 있습니다.

💌 관리와 피드백

관리(Dom)가 피드백과 선물 등의 내용을 작성합니다. 기록(Sub)은 내용을 확인합니다.

📅 기록실

달력을 이전·다음 달로 이동하고 날짜를 누르면 그날의 약속, 루틴, 컨디션, 기록, 피드백을 확인할 수 있습니다.

🗑️ 기록 삭제와 복구

Dom이 기록을 삭제하면 Sub에게 삭제 날짜와 시간이 표시됩니다. 삭제 원본은 30일 동안 보관되며 Dom이 기록실에서 복구할 수 있습니다.

📷 오늘의 순간

일상 사진을 한곳에서 확인하고 원하는 날짜의 기록으로 이동할 수 있습니다.

💕 우리의 기념일

생일, 여행, 데이트 등 기념일을 등록할 수 있으며 대표 기념일은 ‘함께한 지 N일’ 요약에 반영됩니다.

🎨 개인 테마

내 계정의 테마 메뉴에서 라벤더·블라썸·오션·포레스트·크림 중 하나를 선택할 수 있습니다.

💞 우리의 공용 테마

관리(Dom)가 공용 테마를 켜면 같은 공간의 두 사용자에게 동일한 색상 테마가 실시간 적용됩니다.

🌙 화면 표시 방식

라이트·다크·시스템 따라가기를 개인별로 선택할 수 있습니다. 공용 색상 테마와 화면 밝기 설정은 서로 독립적으로 동작합니다.

🗂️ 데이터 관리

내 계정에서 개인 데이터 또는 Room 전체 데이터 삭제 요청을 보낼 수 있습니다.

🛡️ 요청 검토

삭제는 즉시 실행되지 않고 관리자가 검토합니다. 처리 전에는 요청을 취소할 수 있고, 승인·거절 메시지를 확인할 수 있습니다.

🚪 로그아웃

로그아웃해도 서버에 저장된 기록은 삭제되지 않습니다. 같은 계정으로 다시 로그인하면 연결된 공간을 불러옵니다.

두 사용자가 같은 공간에 연결되어 있는지 확인하세요. 초대코드가 다르면 서로의 기록이 보이지 않습니다.
오늘의 약속은 관리(Dom)가 만들고 기록(Sub)이 수행합니다. 나의 루틴은 기록(Sub)이 스스로 만들고 관리합니다.
각 사용자가 이전에 저장한 개인 테마로 즉시 돌아갑니다.
기록실에서 날짜를 선택하면 그날의 약속, 루틴, 컨디션, 기록과 사진을 다시 볼 수 있습니다.
초대코드는 24시간 동안만 유효합니다. 관리(Dom)에게 새 코드를 요청하세요.
우리의 공간 메뉴에서 이전 공간 목록을 열고 원하는 공간으로 전환할 수 있습니다.

📨 문의·고객센터

앱 사용 중 불편한 점이나 데이터 오류를 보내면 관리자가 확인한 뒤 이 화면으로 답변합니다.

비밀번호, 인증번호, 주민등록번호, 금융정보 등 민감한 개인정보는 적지 마세요.
현재 로그인 계정과 Room 정보가 확인용으로 함께 전달됩니다.

    삭제 요청 전 확인
    • 요청 즉시 데이터가 삭제되지 않습니다.
    • 운영자가 사유와 삭제 범위를 확인한 뒤 처리합니다.
    • 두 사람이 함께 사용하는 Room 기록은 상대방의 권리도 함께 검토합니다.
    • 승인 후에도 실제 삭제 전 안내 기간이 있을 수 있습니다.
