# STEP5.10.12 History Center UI QA

1. Light/Dark/System mode에서 기록실 캘린더 배경과 날짜 셀이 같은 테마인지 확인
2. 기록 있는 날짜, 오늘, 선택 날짜의 구분 확인
3. 사진 0/1/2/3/4장 이상일 때 사진 카드 높이와 미리보기 수 확인
4. 사진 카드 클릭 시 전체 사진 팝업 확인
5. 삭제 기록이 있을 때 기록실에는 한 줄 카드만 표시되는지 확인
6. 삭제 카드 클릭 시 중앙 팝업에서 상태 필터·확인·복구 확인
7. PC, Android, iPhone, Fold, Tablet 화면 확인
8. Dom/Sub 계정에서 기존 기록 상세·삭제·복구 기능 회귀 확인
