# HearMe2nite RC2.16.1 PC Layout Optimization

## 목표
PC 화면에서 모바일 확대형처럼 보이는 문제를 줄이고, 넓은 화면에서 카드와 섹션이 더 안정적으로 보이도록 CSS 전용 레이아웃 레이어를 추가했습니다.

## 변경 범위
- `css/style.css`만 변경
- Firebase / Room / Invite / Presence / History JS 변경 없음
- HTML 구조 변경 없음

## 주요 변경
- PC 전용 2열 카드 흐름 적용
- 상단 헤더/버전/온라인 표시 여백 정리
- 카드 그림자와 hover 미세 조정
- 채팅 영역 PC 폭 확장 및 입력창 정리
- 기록실 모달 PC 폭 확장
- 캘린더 셀 크기 PC 기준 미세 조정

## QA
- PC에서 홈 카드가 2열로 자연스럽게 흐르는지 확인
- 모바일/폴드/태블릿 화면이 기존과 크게 달라지지 않는지 확인
- Presence 온라인 점 유지 확인
- 기록실/우리의 공간/오늘의 약속/하루 기록 팝업 정상 동작 확인
