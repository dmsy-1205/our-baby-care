# RC2.15 History UX STEP1

## 변경 내용

- 기록실 캘린더 월 이동 기능 추가
  - 이전 달 / 다음 달 / 오늘 버튼
  - 기록이 없는 달도 확인 가능
- 우리의 기념일을 직접 등록 방식으로 정리
  - 첫날 만난 날 / 자동 D+ 계산 화면 제거
  - 등록한 기념일만 캘린더 아이콘으로 표시
- 기록실 하단 통계 카드 삭제
  - 총 기록일 / 사진 기록 / 미션 기록 / 최근 기록 숨김
- 기록실 패널 구조 정리
  - 캘린더 → 우리의 기념일 → 검색/필터 → 사진 모아보기 → 선택 날짜 기록

## 변경하지 않은 것

- Firebase 데이터 구조
- Room / Invite / Restore
- History 저장 경로
- 하루 기록 상세 팝업
- Presence 온라인 표시
