# RC2.14.3 Safe Integrated Release Notes

이번 배포본은 RC2.12.2 정상본을 기준으로 홈 UX 관련 안정 수정만 병합한 배포용 FULL 버전입니다.

## 주요 반영

1. 오늘의 약속
   - 나만의 루틴/맞춤 루틴 문구를 오늘의 약속으로 변경
   - 오늘의 요약 바로 아래에 배치
   - 기존 오늘의 미션 홈 카드는 숨김 처리

2. 오늘의 요약
   - 큰 카드형 요약을 얇은 아이콘 상태바로 변경
   - 5칸 동일 폭 고정
   - 내용이 있어도 옆으로 밀리지 않도록 수정
   - 수분은 ML 제거, 체중은 kg 제거
   - 없음은 - 로 표시

3. 요약 모달
   - 오늘의 요약 클릭 시 읽기 전용 중앙 모달 표시
   - 수분, 체중, 오늘의 약속, 다음 기념일, 최근 업데이트 표시

## 제외

- RC2.13 Room Presence 계열은 초기 화면 멈춤 이슈로 제외했습니다.

## 검수

- JS 문법 검사 통과
- Firebase 구조 변경 없음
- 기존 Room/History/Custom Routine 데이터 구조 변경 없음
