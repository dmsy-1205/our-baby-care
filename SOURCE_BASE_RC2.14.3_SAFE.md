## 포함 기능

- History 2.0
- 기록실 캘린더 상단 고정
- 날짜 클릭 시 하루 기록 중앙 모달
- 오늘의 미션 홈 카드 숨김
- 나만의 루틴/맞춤 루틴 → 오늘의 약속 문구 정리
- 오늘의 약속 홈 위치 이동
- 오늘의 요약 아이콘 상태바
- 요약 5칸 동일 폭 고정
- 수분/체중 단위 축약 표시
- 없음 → - 표시
- 오늘의 요약 클릭 시 읽기 전용 중앙 모달
