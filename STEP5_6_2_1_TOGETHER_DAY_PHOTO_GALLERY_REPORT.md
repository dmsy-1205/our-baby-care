# HearMe2nite v1.0 STEP5.6.2.1 변경 보고서

## 기준 버전
- HearMe2nite v1.0 STEP5.6.2.0 Full Source

## 변경 사항
1. 오늘의 요약의 `다음 기념일`을 조건부 `함께한 지 N일`로 변경했습니다.
2. 대표 날짜(처음 만난 날)가 등록된 Room에서만 해당 요약 항목이 표시됩니다.
3. 사진 모아보기를 기록실 내부의 펼쳐진 목록에서 작은 카드형 진입 UI로 변경했습니다.
4. 카드 클릭 시 중앙 갤러리 모달이 열립니다.
5. 최신 사진 9장을 먼저 표시하고, 더 보기 시 9장씩 추가합니다.
6. 사진 클릭 시 기존 하루 기록 상세 팝업을 그대로 사용합니다.
7. Firebase 데이터 구조와 데이터 관리 기능은 변경하지 않았습니다.
