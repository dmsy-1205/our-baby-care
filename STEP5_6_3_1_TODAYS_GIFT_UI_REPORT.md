# HearMe2nite v1.0 STEP5.6.3.1 변경 보고서

## 작업명
오늘의 선물 UI 개편

## 변경 내용
- 기존 홈 카드의 `보상 / 휴식` 명칭을 `오늘의 선물`로 변경했습니다.
- 기존 한 개의 통합 카드를 유지하여 홈 화면 길이가 늘어나지 않도록 했습니다.
- 중앙 모달에서 다음 두 가지 감성형 선택 카드를 제공합니다.
  - `🎁 작은 보상`
  - `🌙 편안한 휴식`
- 각 선택 카드에 짧은 설명을 추가하고 선택 상태를 시각적으로 강화했습니다.
- 입력 영역의 명칭을 `오늘의 메시지`로 정리했습니다.
- 기록실 및 최종 완성·복사 화면의 제목을 `오늘의 선물`로 통일했습니다.

## 데이터 호환성
- 기존 `dailyChoice` 값인 `reward`, `rest`를 그대로 사용합니다.
- 기존 `rewardNote` 저장 경로를 그대로 사용합니다.
- Firebase 데이터 구조와 Rules는 변경하지 않았습니다.
- 이전 기록도 새 UI에서 그대로 표시됩니다.

## 제외 사항
- 데이터 삭제 기능
- 푸시 알림
- 새 서버 로직
- Cloud Functions 변경
