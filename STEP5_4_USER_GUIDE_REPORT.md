# HearMe2nite v1.0 STEP5.4 사용설명서 개선 보고서

## 목적
처음 사용하는 사용자가 앱의 핵심 흐름을 쉽게 이해하고, 특히 핵심 기능인 `오늘의 약속`을 놓치지 않도록 사용설명서를 재구성했다.

## 변경 내용
- 홈 가이드에 사용 순서(공간 연결 → 약속 확인 → 하루 기록) 추가
- `오늘의 약속`을 독립 탭과 강조 카드로 추가
- 오늘의 약속 생성·확인·기록 과정을 3단계로 설명
- 관리(Dom)과 기록(Sub)의 역할 설명 추가
- 실제 활용 예시 추가
- `우리의 공간` 설명에서 약속 기능을 분리하고 상대 연결 중심으로 정정
- 검색 예시 및 하단 안내 문구 개선
- 모바일 2열/1열 반응형과 터치 가독성 개선

## 영향 범위
- index.html 사용설명서 마크업
- css/style.css 사용설명서 전용 스타일
- Firebase, Authentication, UID, Room, Invite, Presence, 기록 데이터 로직 변경 없음
