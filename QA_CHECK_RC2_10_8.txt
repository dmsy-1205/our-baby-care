RC2.10.8 QA 체크리스트

1. 기록실 열기
2. 기록이 있는 날짜 클릭
   - 기존 기록 카드가 정상 표시되는지
   - 아래에 기념일 카드가 함께 표시되는지
3. 기념일만 있고 기록이 없는 날짜 클릭
   - 기록 없음 안내가 보이고
   - 아래에 기념일 카드가 표시되는지
4. 처음 만난 날 클릭
   - 처음 만난 날 카드와 D-day가 표시되는지
5. 자동 D+100/D+200/D+300/D+365/D+500 날짜 클릭
   - 자동 계산 기념일이 표시되는지
6. Desktop / Tablet / Fold / Mobile에서 카드 폭과 스크롤 확인
7. 기존 기록 불러오기 / 삭제 / 복사 기능 정상 확인
