# RC2.17 Desktop Dashboard STEP1

## 목적
PC 화면을 모바일 확대형이 아니라 데스크톱 대시보드처럼 보이도록 정리합니다.

## 변경 범위
- CSS 전용 변경
- Firebase / Room / Invite / Presence / History JS 로직 변경 없음
- 1200px 이상 PC 화면에만 적용
- 모바일 / 폴드 / 태블릿 기존 레이아웃 유지

## 주요 변경
- PC 홈을 3열 대시보드 구조로 재배치
- 왼쪽: 오늘의 약속 / 기록
- 가운데: 컨디션 / 관리와 피드백
- 오른쪽: 오늘의 요약 / 실시간 한줄 톡 / 마무리 버튼
- 기록실 모달 PC 화면에서 캘린더와 타임라인 2열 배치

## QA
- PC 홈 카드 배치 확인
- 온라인 점 유지 확인
- 우리의 공간 열기/닫기 확인
- 기록실 캘린더/타임라인 확인
- 모바일 화면은 별도 실제 기기 테스트 예정
