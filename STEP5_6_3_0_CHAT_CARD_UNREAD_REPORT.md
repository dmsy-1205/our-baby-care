# HearMe2nite v1.0 STEP5.6.3.0 변경 보고서

## 적용 항목
- 기존 상시 노출 채팅창을 홈 카드형으로 변경
- 오늘의 요약 아래에 채팅 카드 배치
- 카드 클릭 시 기존 크기의 중앙 채팅 모달 표시
- 최근 메시지 한 줄 미리보기 및 전송 시간 표시
- 상대방 온라인/오프라인 상태 표시
- 사용자별 읽지 않은 메시지 숫자 배지 추가
- 채팅 모달을 열면 현재 메시지까지 읽음 처리
- 오프라인 상대에게 보낸 메시지는 저장된다는 안내 표시

## 데이터 구조
- 기존 메시지 경로 유지: rooms/{roomCode}/messages
- 읽음 상태 추가: rooms/{roomCode}/chatReadStatus/{uid}
- Firebase Rules 변경 없음: 기존 Room 멤버 권한 범위 안에서 저장

## 제외 항목
- 푸시 알림
- 메시지 수정/삭제
- 사진 및 파일 전송
- 타이핑 상태

## 안정성
- 인증, Room, 기록, 데이터 관리 구조 변경 없음
- 최근 메시지 50개 구독 방식 유지
