# HearMe2nite v1.0 STEP6.1.7 QA

## 목적
관리자 페이지가 권한 확인 대기 화면에 멈추지 않고 로그인 세션 확인 직후 표시되도록 수정.

## 변경
- 관리자 로그인 세션이 있으면 Admin Shell 즉시 표시
- Firebase Rules가 사용자/Room/삭제요청 데이터 접근을 최종 검증
- 데이터 조회 실패 시 대시보드는 유지하고 상단에 오류 표시
- Auth 응답 무한 대기 방지

## 확인
1. 관리자 계정 로그인
2. 내 계정 > 운영 콘솔
3. Admin Dashboard 즉시 표시 확인
4. 사용자/Rooms/Recovery 데이터 조회 확인
5. 일반 계정은 운영 콘솔 버튼이 보이지 않는지 확인
